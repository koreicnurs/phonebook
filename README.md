# phonebook
Simple phonebook


## Установка

```
1. Склонируйте репозитории
    git clone https://github.com/koreicnurs/phonebook.git

2. Войдите в папку с проектом
    cd django-phonebook/

3. Установите виртуальное окружение
    virtualenv --python `which python3` env

4. Активируйте виртуальное окружение
    . env/bin/activate
    
6. Установите все компоненты проекта
    pip install -r requirements.txt

7. Сделайте миграцию
    ./manage.py migrate

8. Создайте супер юзара для админки
    ./manage.py createsuperuser

9. Запустите сам проект
    ./manage.py runserver

10. Создайте себе логин для начала работы

```
