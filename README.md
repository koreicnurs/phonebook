# phonebook
Simple phonebook


## Установка

```
1. Склонируйте репозитории
    git clone https://github.com/koreicnurs/phonebook.git

2. Войдите в папку с проектом
    cd django-phonebook/

3. Установите виртуальное окружение
    virtualenv --python `which python3` env

4. Активируйте виртуальное окружение
    . env/bin/activate
    
5. Установите все компоненты проекта
    pip install -r requirements.txt

6. Сделайте миграцию
    ./manage.py migrate

7. Создайте супер юзара для админки
    ./manage.py createsuperuser

8. Запустите сам проект
    ./manage.py runserver

9. Создайте себе логин для начала работы

```
